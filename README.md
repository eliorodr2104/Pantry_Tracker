# Shelf Management App

This project is a shelf management application that combines barcode scanning with API interfacing to provide detailed product information. It draws inspiration from technologies commonly used in supermarkets but applies them in a household context.

## Key Features

- **Barcode Scanning**: The application allows users to scan product barcodes.
- **API Interfacing**: Product information is obtained through interfacing with APIs, offering detailed specifications.
- **Data Customization**: Users have the ability to customize product information according to their needs.
- **Personal Virtual Shelf**: All scanned and customized product information is integrated into a personal "virtual shelf."
- **Real-Time Database**: An integrated database updates data in real-time on connected devices, enabling efficient and convenient product management.
